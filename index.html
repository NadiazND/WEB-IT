Portofolio & Studi Kasus

Contoh proyek yang menunjukkan pendekatan teknis, keputusan arsitektural, dan hasil bisnis nyata.

RetailX — Platform E‑Commerce

Masalah: monolith tidak skalabel saat puncak kampanye. Solusi: re-architect ke microservices, event-driven order pipeline, dan autoscaling. Hasil: throughput 4x, puncak latency turun 60%, dan cost per order turun 28%.

Teknologi: Kubernetes, Kafka, PostgreSQL, Prometheus, Grafana.

SehatCare — Sistem Manajemen Rumah Sakit

Masalah: proses klinis terfragmentasi dan kepatuhan sulit diaudit. Solusi: implementasi EMR berbasis FHIR, secure API gateway, dan audit trail terpusat. Hasil: waktu proses klinis berkurang 35%, kepatuhan audit meningkat, dan interoperabilitas antar fasilitas terwujud.

Teknologi: FHIR, OAuth2, ELK stack, PostgreSQL.

FinAnalytics — Platform Analitik Real-time

Masalah: deteksi anomali transaksi lambat. Solusi: streaming pipeline, feature store, dan model anomali terintegrasi ke dashboard operasional. Hasil: deteksi anomali 10x lebih cepat dan false positive menurun 40%.

Teknologi: Kafka, Flink, Redis, MLflow, Superset.

SmartCity — Platform Manajemen Kota Pintar

Masalah: data sensor tersebar dan sulit dimanfaatkan. Solusi: integrasi IoT, pipeline real-time, dan dashboard layanan publik. Hasil: respons insiden lebih cepat, optimasi resource, dan peningkatan kepuasan warga.

Teknologi: MQTT, TimescaleDB, Grafana, Kubernetes.

Karir

Kami mencari talenta yang berorientasi solusi, memiliki ownership, dan ingin berkembang dalam lingkungan engineering yang menantang.

Software Engineer (Backend)

Tanggung jawab: desain API, microservices, dan optimasi performa. Kualifikasi: 3+ tahun, pengalaman dengan K8s, CI/CD, dan testing otomatis.

Benefit: remote-first, learning budget, dan mentorship.

Data Engineer

Tanggung jawab: pipeline ETL/ELT, streaming, dan data quality. Kualifikasi: pengalaman Spark/Kafka, data modeling, dan observability.

Benefit: exposure ke ML ops dan proyek skala nasional.

DevOps / SRE

Tanggung jawab: IaC, monitoring, incident response, dan reliability engineering. Kualifikasi: Terraform, Prometheus, K8s, dan runbooks.

Benefit: on-call rotation yang adil, career path SRE.
