Portofolio & Studi Kasus

Beberapa proyek terpilih yang menunjukkan pendekatan teknis dan hasil bisnis yang nyata.

RetailX — Platform E‑Commerce

Re-architect monolith ke microservices, event-driven order pipeline, observability. Hasil: throughput 4x, latency puncak turun 60%.

SehatCare — Sistem Manajemen Rumah Sakit

Implementasi EMR berbasis FHIR, secure API gateway, audit trail. Hasil: proses klinis lebih efisien dan kepatuhan meningkat.

FinAnalytics — Platform Analitik Real-time

Streaming pipeline, model anomali, dan dashboard operasional. Hasil: deteksi anomali 10x lebih cepat.

SmartCity — Platform Manajemen Kota Pintar

Integrasi sensor IoT, pipeline data real-time, dan dashboard operasional untuk layanan publik. Hasil: respons insiden lebih cepat, optimasi resource, dan peningkatan layanan warga.
